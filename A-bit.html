BatCom

开源可视化拖拽编译工具，轻松自制编程语言

项目介绍

BatCom 是一款在 GitHub 完全开源的编译制作软件，无需手写大量代码，通过可视化拖拽模块，就能快速打造属于自己的简易编程语言，一键打包生成可运行程序。

软件特色

  • 纯可视化拖拽操作，零基础也能自定义语法
  • 开源免费，全部代码托管于 GitHub
  • 轻量化编译内核，支持导出 bat / exe 程序
  • 内置语法调试窗口，实时预览运行效果
  • 无捆绑、无广告，本地离线运行
